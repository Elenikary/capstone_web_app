/About This Project/


We are proposing the design and assembly of a small color detecting device for beginner painters and educators.
This product aims to help people who are interested in learning about color theory and how to practically
blend colors to achieve the desired hue of a color. The sensor will read the color of different surfaces
(e.g. on walls, objects, other images etc.) in order to help determine the color code of the color detected and
therefore propose to the user ways and methods for them to reproduce the observed color using their color paints
by sending the data collected from the color sensor to a web application. Using an RGB sensor we will be able to
detect the color observed by the sensor into combinations of primary colors in order to help the user determine what
mixtures they may need to do in order to come up with the desired color. Our product will consist of a number of different
sensors that will work together to enhance the performance of the device and make it more accurate. Some of these
parts/sensors are: a touchscreen interface / screen display, a distance sensor (to detect how close the device is
from the object), a sensor that detects whether or not the device is on the observed surface, a motion sensor that helps
the user “wake the device up” by moving it etc. We will also use an ESP8266 microchip to establish the wifi connection
with the web application.
